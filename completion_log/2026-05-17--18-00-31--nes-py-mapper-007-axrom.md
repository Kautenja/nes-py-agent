# nes-py Mapper 007 AxROM Implementation

## Summary

- Added native mapper 007 / AxROM support in `nes-py` with one switchable 32 KiB PRG window, mapper-owned CHR RAM, and one-screen lower/upper mirroring from register bit 4.
- Registered mapper 7 as `AxROM` in the native mapper factory so `_native.is_mapper_supported(7)` and `NESEnv` validation accept it after the implementation is wired.
- Modeled bus conflicts for NES 2.0 submapper 2 while leaving default iNES/submapper 0 AxROM writes as no-conflict.
- Added dedicated native Catch2 coverage for PRG banking, bank masking, one-screen mirroring, CHR RAM, NES 2.0 bus-conflict resolution, and emulator backup/restore state.
- Added public Python mapper 007 tests, including the representative `Battletoads (USA)` fixture path at `nes_py/tests/games/battletoads.nes`.
