# nes-py Mapper 009 MMC2

## Summary

- Added native mapper 009 / MMC2 support in `nes-py` with one switchable 8 KiB PRG bank at `$8000-$9fff`, three fixed 8 KiB PRG banks at `$a000-$ffff`, mapper-controlled mirroring, and no IRQ path.
- Implemented MMC2 CHR latch behavior for PPU reads at `$0fd8`, `$0fe8`, `$1fd8-$1fdf`, and `$1fe8-$1fef`, with latch changes staged from the PPU address observer and applied after the triggering CHR byte is fetched.
- Registered mapper 9 in `MapperFactory` / `IsMapperSupported`, which allows Python `NESEnv` validation to accept mapper 9 only once the native mapper is available.
- Added dedicated native Catch2 coverage for PRG bank masking, mirroring, exact CHR latch addresses/ranges, non-latch neighbors, initial FE latch selection, and emulator backup/restore.
- Added Python public mapper 009 smoke coverage, including the local `Mike Tyson's Punch-Out!! (USA)` fixture path at `nes_py/tests/games/mike-tysons-punch-out.nes`.
